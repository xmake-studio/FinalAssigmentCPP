# Расписание (вариант 12)

Итоговое задание по C++. Консольная программа для работы с базой данных
расписания учебных занятий.

## Поля записи

| #  | Поле           | Тип                     |
|----|----------------|-------------------------|
| 1  | id             | `int` (ключевое)        |
| 2  | Группа         | `std::string`           |
| 3  | Предмет        | `std::string`           |
| 4  | Преподаватель  | `std::string`           |
| 5  | День недели    | `unsigned char` (1..7)  |
| 6  | Номер пары     | `unsigned char` (1..4)  |

## Сборка

CMake:

```
cmake -S . -B build
cmake --build build --config Release
```

g++ / MinGW:

```
g++ -std=c++17 -O2 -Wall -Wextra src/*.cpp -o schedule_db
```

MSVC:

```
cl /std:c++17 /EHsc /utf-8 /W4 src\main.cpp src\Menu.cpp src\ScheduleDB.cpp src\ScheduleRecord.cpp /Fe:schedule_db.exe
```

## Запуск

```
schedule_db                     # читает data/schedule.csv
schedule_db path/to/file.csv    # явный путь
```

## Меню

1. Загрузить БД из файла
2. Сохранить БД в файл
3. Показать все записи (таблица)
4. Добавить запись
5. Удалить запись по ID
6. Сортировать по ID
7. Поиск по ID (бинарный)
8. Выборка по диапазону дней недели
9. Проверить накладки в расписании
10. Показать расписание группы
0. Выход

## Файл данных

`data/schedule.csv` — 29 записей, разделитель `;`, кодировка UTF-8.
В файле специально оставлено несколько накладок, чтобы пункт 9 имел
что показывать.
